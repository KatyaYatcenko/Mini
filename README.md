
# Асистент відповідей у Telegram (Draft Generator)

## Принцип роботи (важливо!)

1. Приходить повідомлення в Telegram.
2. ШІ аналізує контекст і автоматично обирає один із режимів спілкування (`Colleague` — робочий, `Friend` — дружній, `Lover` — близький).
3. ШІ одразу готує **чернетку** відповіді відповідно до обраного режиму та мови співрозмовника.
4. З'являється вікно з текстом вхідного повідомлення і чернеткою.
5. Людина обирає:
   - **ВІДПРАВИТИ** — надсилає чернетку як є в один клік;
   - **РЕДАГУВАТИ** — текстове поле завжди доступне для правок, можна змінити або дописати текст вручну перед відправкою.
6. Якщо просто закрити вікно — нічого не надсилається.

Свідомо **немає**:
- штучно нав'язаних емодзі та обов'язкових запитань наприкінці кожної фрази;
- фейкових затримок "ніби людина друкує" або навмисних одруківок;
- автоматичної відправки без явного підтвердження користувача.

Інструмент створений для того, щоб **швидше формулювати** адекватні відповіді, зберігаючи природний стиль мовлення та повний контроль над чатом.

## Встановлення

```bash
pip install -r requirements.txt

```

## Налаштування

1. Скопіюй `.env.example` у `.env`:
```bash
cp .env.example .env

```

2. Отримай `TG_API_ID` і `TG_API_HASH` на https://my.telegram.org → API Development Tools.
3. Отримай ключ OpenRouter на https://openrouter.ai/keys і встав у `OPENROUTER_API_KEY`.
4. За потреби зміни `OPENROUTER_MODEL` на іншу модель.

## Запуск

```bash
python main.py

```

При першому запуску Telethon попросить увійти у Telegram-акаунт (номер телефону + код підтвердження) — це стандартна, безпечна авторизація Telegram, дані сесії зберігаються локально у файлі `assist_session.session` і нікуди не відправляються.

## Функціональні особливості

* **Адаптація тону:** Модель самостійно класифікує тип стосунків із користувачем і підлаштовує стиль мовлення. Якщо співрозмовник поводиться токсично чи порушує кордони, ШІ реагує холодно та ставить його на місце.
* **Підтримка Vision (аналіз медіа):** Бот вміє кодувати вхідні зображення в Base64 та передавати їх моделі, щоб чернетка враховувала вміст надісланого фото.
* **Language Matching:** Автоматичне розпізнавання мови повідомлення та відповідь тією ж мовою.

## Обмеження цієї версії

* Реагує тільки на особисті чати (ігнорує групи та канали).
* Одне спливаюче вікно на кожне нове повідомлення; якщо повідомлень багато одночасно, вікна з'являються по черзі.
